# Project overview — BMDM Myeloid-Inflammation (R scripts)

The R counterparts of the Python BMDM analysis: BMDM_analysis_all.R and BMDM_volcano_DEcounts.R
perform the actual edgeR/TMM differential-expression modeling and diagnostics (composition,
logFC tails, cross-axis concordance, CH-gene heatmap) that the Python scripts in
`01_BMDM_Myeloid_Inflammation` (Python portfolio) build on. See that Python project's overview
for the full biological context (clonal burden HB/LB vs lentiviral marking pos/neg).
