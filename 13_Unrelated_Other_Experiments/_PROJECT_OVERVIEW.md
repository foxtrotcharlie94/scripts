# Project overview — Unrelated / Other Experiments

Scripts that, on inspection, belong to different projects entirely rather than the mouse clonal-
hematopoiesis / lentiviral-marking study: a human patient telomere-length vs CH-genotype
regression (`Telomeres.R`), a NHANES epidemiology reference-range analysis (`nhanes.R`), a
brightfield-microscopy cell-confluence image-analysis tool (`Well_coverage_drightfield_
analysis.R`), a mislabeled CDS-size/BED-coordinate extraction script for the genes Kmt2d and
Pkd1 (`Lenti_UMI_cutoff_50percent_pool.R` — despite its Lenti-suggestive filename), and a small
permutation-test script validating that DOX and 5-FU chemotherapy produce comparable
cytoreduction (`permut_cell_death_R'.R`, also duplicated at the Downloads root). Flagged here
rather than folded into the CH narrative, per instructions to describe each file accurately.
