# Project overview — Division-Burden Methylation Clocks (R script)

`run_analysis.R` reproduces, end-to-end in R, the Python pipeline that tests whether forced HSC
proliferation (10-cell transplant reconstitution) produces DNA-methylation changes resembling
natural aging, using the public GSE44117 RRBS dataset and three published mouse epigenetic
clocks (Blood, WLMT, YOMT). See the Python portfolio's `02_Division_Burden_Methylation_Clocks`
project overview for full background.
